# ForgeCare Medical Center - Hospital Platform

A modern, responsive hospital management and patient booking platform built with vanilla JavaScript and Tailwind CSS.

## Overview

This is a comprehensive hospital intelligence platform designed for multi-branch medical centers across Bangladesh. It provides appointment booking, doctor discovery, department information, and emergency services management through an intuitive web interface.

## Features

- **Multi-Branch Management** - Support for hospitals across Dhaka, Chattogram, Sylhet, and Rajshahi
- **Doctor Directory** - Browse doctors by specialization with ratings, experience, and availability
- **Appointment Booking** - Schedule consultations with flexible time slots and consultation modes (in-person, telemedicine, home-visit)
- **Department Explorer** - Explore 8+ medical departments with bed capacity and services
- **Emergency Services** - Quick access to emergency hotline and urgent care information
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Real-time Notifications** - Toast notifications for user actions and confirmations

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Tailwind CSS (utility-first), Custom CSS animations
- **Architecture**: Component-based rendering with vanilla JavaScript
- **Data**: Static data structure with real-time state management

3. Browse available doctors and their specializations
